# Family Fashion 👗🛍️

Family Fashion is a full-stack e-commerce web application developed using Flask, HTML, CSS, JavaScript, and MySQL. It provides a seamless online shopping experience with user authentication, product browsing, shopping cart, wishlist, and order management.

## Features

- User Registration & Login
- User Profile & Settings
- Browse Products by Category
- Search Products
- Shopping Cart
- Wishlist
- Quantity Update
- Secure Password Hashing
- Place Orders
- Order History
- Responsive User Interface

## Technologies Used

## Installation

1. Clone the repository

bash
git clone https://github.com/Seenupriya1/E-Commerce-Website.git


2. Navigate to the project folder

bash
cd E-Commerce-Website


3. Install the required packages

bash
pip install flask flask-mysqldb werkzeug


4. Create the MySQL database using the database.sql file.

5. Update your MySQL credentials in app.py.

6. Run the application

bash
python app.py
